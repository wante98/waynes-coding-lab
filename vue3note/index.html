Vue.js 3 課程筆記

前置觀念

ES6 語法
ECMAScript 6 (簡稱 ES6)，是 JavaScript 語言新一代的標準，在 2015 年 6 月正式發佈。
跟以前的 Javascript 有以下差別
作用域 Scope

箭頭函式 ArrowFunction

新增變數宣告 let const

解構 Deconstructor

Node.js: Javascript 運行在伺服器端的執行環境 Runtime

SPA 架構：
SPA （Single-Page Application） 顧名思義，就是整個網站實際上其實只有一個頁面。 為了解決上述提到的頁面重整的問題，我們可以使用 AJAX （Asynchronous JavaScript And XML，非同步的 JavaScript 與 XML） 的技術來處理，令其在送出資料且收到伺服器的回覆後，只針對需要更新的頁面元素來更新

傳統是使用 MPA，一個畫面，一個 HTML 檔案

Vue.js 相關介紹

Vue.js 是一個開源 MVVM 的 Javascript 框架，也算是 SPA(單頁應用程式)

為何要使用框架進行前端開發?

(1)舊有技術操作 DOM 稍嫌臃腫

(2)Component 能增加前端開發、維護效率

Vue.js 運行原理

(vite.config or vue.config)->app.vue->main.js->index.html

Vue.js 生態系

Vuex：狀態管理庫(類似 React 的 Redux)

Pinia：最新一代的 Vuex 狀態管理庫，相對於 Vuex 移除 Mutation 的部分

Vue CLI：可以在本機建立 Vue 專案，不需要在每個頁面引入 VUE 的 CDN

Vue Router：路由設定

Axios：串接 API 的套件

Vue 2 與 Vue 3 的差別

(1) Vue 3 不支援 IE

(2) Vue 3 支援 Composition API，相較於 Vue 2 Option API，script 裡面函式邏輯會比較單純，不會那麼雜

(3) View Design 的 div slot=’header’要改成 template #header

(4) h 函式渲染方式有改變
