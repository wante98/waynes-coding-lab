如何從菜鳥軟體工程師進化成資深工程師?

網站上很多人分享自己的轉職經驗，程式補習班也會過度吹捧程式轉職成功，而忽略失敗的人，其實也不在少數。這邊筆者想要從自己的經驗分享如何菜鳥軟體工程師進化成資深工程師

1.加強程式基本功

當我們轉職成功後，其實面臨到的是諸多專案在死限下交件的壓力。又常常在還沒有很了解一們技術的狀況下就被丟去開發或是解 issue。在這樣的情況程式的基本功就很重要。當基本功紮實，面對新的技術就能以比較快的能力上手。這樣遇到 bug 比較容易有線索解決。

寫部落格是一個很好檢視自己能力得機會，建議可以寫技術部落格，並更新自己的資訊

2.學會從解決問題進化到提出問題，提出解決方案並實行

剛入門軟體開發時，通常遇到的情境比較像是 PM 提出一個 UI 跑版要調整，或是這個 API 打過去顯示 404 需要排除。我們在解決問題的途中，要不斷思考這個問題是如何產生?我要如何去發現並解決這個問題

3.心態 正向思考

即便是遇到許多專案上的困難而失落，不要氣餒。可以維持正向思考是很重要了。讓自己有時候喘口氣，散散步、跟朋友聊聊天。說不訂 bug 就突然有頭緒解了

總而言之，即使入門後，後續的成長與學習是很重要的。把握做好專案，把每一份事情好好做好，轉職工程師仍然大有可為。
